# Journal Calendar

A journaling app built with SwiftUI that combines a day calendar with rich event entries.

## Features

- **Day calendar view** with hourly time slots and event blocks
- **Create, edit, and delete** event blocks with titles and time ranges
- **Drag and drop** events to reschedule with 15-minute snapping
- **Pinch to zoom** the timeline in and out
- **Sub-block content types** within each event:
  - Journal text entries
  - Image uploads with full-screen viewer
  - Links
  - Location with map preview and search
- **Overlapping event layout** with automatic column positioning
- **Day navigation** with previous/next controls
